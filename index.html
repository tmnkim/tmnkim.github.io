CSS Tables Cheatsheet

This is a quick reference sheet for styling tables using a cascading style sheet (CSS)

CSS table attributes
Tag Description Example
border Sets the border properties table, th, td {
border: 1px solid black;
}
width If you need a table to span the width of the screen width: 100%;
border-collapse Sets whether the table borders should be collapsed to a single border border-collapse: collapse;
padding Default table cell padding is 2px padding: 0.5rem;
border-spacing Sets the distance between borders of adjacent cells. Applies only when border-collapse is set as separate. border-spacing: 2px;
